# Arkanoid

Игра на C++ с использованием библиотеки SFML 3.

## Платформа

Windows 10/11 (MSYS2 / UCRT64)

## Зависимости

- [MSYS2](https://www.msys2.org/) — окружение для сборки
- [SFML 3.x](https://www.sfml-dev.org/) — графика, окно, система
- GCC 16+ (устанавливается через MSYS2)

## Установка зависимостей

1. Скачать и установи [MSYS2](https://www.msys2.org/)

2. Открыть **MSYS2 UCRT64** (из меню Пуск) и выполнить:

```bash
pacman -Syu
pacman -S mingw-w64-ucrt-x86_64-gcc mingw-w64-ucrt-x86_64-sfml mingw-w64-ucrt-x86_64-cmake mingw-w64-ucrt-x86_64-make
```

3. Добавь `C:\msys64\ucrt64\bin` в системный PATH:
   - Пуск → «Изменение переменных среды» → Path → Добавить

## Сборка

Открой **MSYS2 UCRT64**, перейди в папку с проектом и выполни:

```bash
cd /путь/к/arkanoid
cmake -B build -G "MinGW Makefiles" -DCMAKE_MAKE_PROGRAM=mingw32-make -DCMAKE_CXX_COMPILER=g++
cmake --build build
```

## Запуск

```bash
./build/arkanoid.exe
```

## Управление

| Клавиша | Действие |
|---------|----------|
| ← →     | Движение ракетки |
| Space   | Отпустить шарик (после прилипания) |
| P       | Пауза / продолжить |
| R       | Рестарт после поражения |
| Esc     | Выход |

## Механики

- Блоки с HP от 1 до 4 — цифра показывает оставшиеся жизни
- Неразрушаемые серые блоки
- Блоки-ускорители (фиолетовые)
- Бонусы в виде падающих ромбиков:

| Цвет | Бонус |
|------|-------|
| Фиолетовый | Расширение ракетки |
| Розовый | Сужение ракетки |
| Светло-розовый | Ускорение шарика |
| Голубой | Замедление шарика |
| Тёмно-фиолетовый | Липкий шарик (отпустить — Space) |
| Синий | Дополнительный шарик |

- Два шарика могут отталкиваться друг от друга (бонус 9)
- Счёт: +1 за каждое попадание, −3 за промах
- 3 промаха — конец игры
